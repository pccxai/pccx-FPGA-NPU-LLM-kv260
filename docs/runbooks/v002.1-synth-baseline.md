# v002.1 Synth Baseline Recording

This runbook records the post-synthesis timing and utilization baseline for
follow-up review. It does not run implementation, generate a bitstream, touch a
KV260 board, or make a closure claim.

## Output

The recorder writes machine-readable JSON here:

```text
hw/build/reports/synth_baseline.json
```

The JSON includes:

- WNS, TNS, WHS, and THS.
- Total DSP, BRAM36-equivalent, URAM, LUT, and FF utilization.
- Top setup critical paths plus the worst hold path available from Vivado.
- Source-derived instance totals for each `*_pkg.sv` parameter group.
- Source report paths for the generated post-synth timing and utilization
  reports.

`hw/build/` is ignored by git. Do not commit the generated Vivado project,
reports, checkpoints, or bitstreams.

## Normal Flow

Run synthesis from a user terminal:

```bash
cd hw
PCCX_VIVADO_JOBS=1 ./vivado/build.sh synth
```

`build.sh synth` sources `vivado/scripts/record_synth_baseline.tcl`
automatically after `vivado/synth.tcl` completes. When the command returns,
check:

```bash
jq . build/reports/synth_baseline.json
```

If `jq` is not installed, use any JSON reader or inspect the file directly.

## Manual Flow

If synthesis was already run in an interactive or custom Vivado session, open
the synthesized run and source the recorder:

```tcl
open_project hw/build/pccx_v002_kv260/pccx_v002_kv260.xpr
open_run synth_1 -name synth_1
source hw/vivado/scripts/record_synth_baseline.tcl
```

When running from inside `hw/`, use the relative source path instead:

```tcl
source vivado/scripts/record_synth_baseline.tcl
```

## Evidence Path

For issue or PR follow-up, cite:

```text
hw/build/reports/synth_baseline.json
hw/build/reports/timing_summary_post_synth.rpt
hw/build/reports/utilization_post_synth.rpt
```

Record the JSON values in the review text instead of committing ignored build
artifacts. Use wording such as "post-synthesis baseline recorded" and keep
implementation, bitstream, board, throughput, and closure statements gated on
their own evidence.
